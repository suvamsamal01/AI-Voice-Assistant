# 🤖 AI Voice Assistant

A powerful desktop AI Assistant built with Python, Ollama, Llama 3.2, LLaVA, Speech Recognition and CustomTkinter.

## ✨ Features

- 🎤 Voice Commands
- 🗣️ Text-to-Speech Response
- 🤖 AI Chat using Ollama + Llama 3.2
- 🖼️ Image Analysis using LLaVA
- 🌦️ Weather Information
- 🎵 YouTube Music Search
- 🧮 Basic Math Calculations
- 🔋 Battery Monitoring
- 💻 CPU Usage Monitoring
- 📸 Screenshot Capture
- 📂 Open Applications with Voice
- 🎙️ Wake Word Detection ("Hey Jarvis")
- 🌙 Modern CustomTkinter UI
- ⚡ Fast Local AI Processing

## 🛠️ Tech Stack

- Python
- Ollama
- Llama 3.2
- LLaVA
- CustomTkinter
- SpeechRecognition
- Pyttsx3
- Requests
- Psutil
- PyAutoGUI
- Pillow

## 🚀 Installation

```bash
git clone https://github.com/suvamsamal01/AI-Voice-Assistant.git

cd AI-Voice-Assistant

pip install -r requirements.txt

python main.py
```
